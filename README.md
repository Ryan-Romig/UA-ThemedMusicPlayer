
# UnderwaterAudioMusicManager
Music Player for Underwater Audio Project

What works:
Play songs
skip to next track
previous track
pause
resume playing
close button
import songs into list
shuffle toggle on/off
repeat
select song
add song to library
volume slider (needs adjusted)
track scrubbing

responsive up to mobile size
