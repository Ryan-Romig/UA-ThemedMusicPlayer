
# UnderwaterAudioMusicManager
Music Player for Underwater Audio Project

What works:
Play songs
skip to next track
previous track
pause
resume playing
close button
import songs into list
shuffle toggle on/off
repeat toggle on/off
select song
add song to library
volume slider (60% scale)
track scrubbing
responsive up to mobile size(500px)
